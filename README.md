# Anvil

A multi-tool workspace for Windows. Kanban boards, whiteboards, diagrams, code editing, and more — all in one dockable desktop app. Everything runs locally. No subscriptions, no cloud, no tracking.

## Tools

- **Kanban Board** — Visual task management with swim lanes, backlog, and archive
- **Excalidraw** — Hand-drawn style diagrams and whiteboards
- **Draw.io** — Professional flowcharts, UML, and network diagrams
- **Mermaid** — Text-based diagram generation
- **Monaco Editor** — VS Code's editor engine, embedded
- **Knowledge Base** — Linked document library with full-text search
- **PDF Viewer** — View documents without leaving the workspace
- **Dashboard** — At-a-glance project summaries
- **Calendar** — Time-based view of project work
- **Document Browser** — Navigate project files in-app
