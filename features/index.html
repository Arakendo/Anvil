    Features

    AnvilKanban is a dockable multi-tool desktop workspace for Windows. Every tool runs locally — no internet required, no accounts needed.

    Project Management

    Kanban Board

    Visual card-based workflow with fully customizable swim lanes.

    • Drag-and-drop cards between lanes
    • Tag system for categorization and filtering
    • Backlog and archive discipline built in
    • Real-time search across all cards

    Backlog

    A calm, searchable inventory of uncommitted work.

    • Browse all uncommitted cards without clutter
    • Filter by tag, date, or keyword
    • Commit cards to active Kanban lanes with one click

    Archive

    Long-term storage for completed work.

    • Move finished cards off the board without deleting them
    • Browse and search historical work
    • Restore archived cards to active lanes

    Drawing & Diagramming

    Excalidraw

    Hand-drawn style diagrams and whiteboards — fast, expressive, easy to share.

    • Freehand drawing, shapes, arrows, and text
    • Export to PNG, SVG, or Excalidraw JSON
    • Offline-capable, no external services

    Draw.io

    Professional diagramming with the full Draw.io shape library.

    • UML, flowcharts, ERD, network diagrams
    • Layers, groups, and complex layouts
    • SVG and PNG export

    Mermaid

    Generate diagrams from code — version control friendly.

    • Flowcharts, sequence diagrams, Gantt charts
    • Live preview as you type
    • Export to SVG

    Content & Code

    Monaco Editor

    The editor engine behind VS Code, embedded directly in your workspace.

    • Syntax highlighting for 100+ languages
    • IntelliSense, autocomplete, multi-cursor
    • Find and replace with regex support

    Knowledge Base

    A searchable, linked document library for capturing and connecting ideas.

    • Create and link documents with Markdown
    • Full-text search across all entries
    • Organize by tags and categories

    PDF Viewer

    View and navigate PDF documents without leaving your workspace.

    • Fast rendering with zoom and search
    • Bookmarks and page navigation
    • Print support

    Workspace

    Dashboard

    At-a-glance summaries of active projects and recent work.

    • Card counts by lane and project
    • Recent activity feed

    Calendar

    Plan work over time with project-linked calendar views.

    • View and schedule Kanban cards by date
    • Milestone tracking

    Document Browser

    Browse and open project files without switching applications.

    • Folder tree navigation
    • Quick-open recent documents
